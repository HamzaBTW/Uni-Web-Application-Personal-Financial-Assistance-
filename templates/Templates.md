# What each page looks like:

## 1. Main Page 

## 2. Login Page

1. Top nav bar with logo + links
2. Left promo panel with feature highlights
3. Auth card with Login ↔ Sign Up tab toggle (click to switch)
4. Login mode: email, password, remember-me checkbox, forgot link
5. Sign Up mode: username, email, password, confirm password
6. Social login buttons, divider, and footer

## 3. Personal Dashboard 

1. Dark sidebar with nav items + user avatar
2. Top bar with search + notifications
3. 4 summary cards: 
   - Net Worth 
   - Income Total,
   - Asset Total,
   - Liabilities Total — each with a colored accent     stripe, value placeholder, and up/down change indicator.
4. Bar chart area + circle/breakdown chart with legend
5. Recent activity table with color-coded status badges (5 rows)

## 4. Same sidebar/topbar layout (Assets item highlighted)

1. "Add New" button in page header
2. Form card with 7 fields:
   - Asset Name,
   - Asset Type (dropdown),
   - Value,
   - Purchase Date,
   - Location/Account,
   - Currency (dropdown),
   - Notes (full-width textarea) — plus Save/Cancel actions
3. Filter bar with search input + 3 filter dropdowns
4. Data table with 7 columns, 6 rows, color-coded type badges (Property, Cash, Stock, Other), and Edit/Delete action buttons per row
5. Pagination controls
